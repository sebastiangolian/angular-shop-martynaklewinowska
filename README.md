# angular-shop-mk
MK Shop

## Getting started
```bash
npm install
```
```bash
npm run start
```
```bash
npm run start-test
```

## Scripts
Build project
```bash
npm run build
```
Build prod project
```bash
npm run build-prod
```
Test project
```bash
npm run test
```
Test e2e project
```bash
npm run e2e
```
Lint project
```bash
npm run lint
```
Deploy files in gh-pages
```bash
npm run deploy
```
